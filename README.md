# Platformer.exe 🎮

Добро пожаловать в проект **Platformer.exe**! Это классический двумерный пиксельный платформер, написанный на Python с использованием библиотеки Pygame. В игре реализована физика прыжков, анимация персонажа, сбор предметов и динамическая загрузка уровней из файлов JSON.

## 🔥 Игровой функционал и особенности

* **Управление:** Движение влево/вправо на клавиши `A` и `D`, прыжок — `Пробел`.
* **Механика игрока:** Реализована гравитация, коллизии (столкновения) с твердыми блоками земли и травы, а также плавная анимация бега. При гибели персонаж превращается в призрака и улетает вверх 👻.
* **Интерфейс и меню:** Полноценное главное меню с кнопками **Start** и **Exit**. Во время игры отображается счетчик собранных алмазов и кнопка **Restart** в случае гибели. На игру дается 5 жизней.
* **Опасности и бонусы:** Игроку нужно собирать алмазы (звучит приятный кастомный звук) и избегать ловушек с водой. Конечная цель — добраться до двери для перехода на следующий этап.
* **Динамические уровни:** Карта мира не прописана жестко в коде, а подгружается из внешних файлов конфигурации (`level1.json`, `level2.json` и т.д.), что позволяет легко создавать новые уровни.

---

## 🚀 Технологии
* **Python 3** — основной язык разработки.
* **Pygame** — игровой движок для работы с графикой, физикой, звуком и управлением.
* **JSON** — формат хранения и загрузки карт уровней.

---

## 📦 Как запустить проект в PyCharm

1. Создайте в PyCharm проект и установите библиотеку через терминал: `pip install pygame`.
2. Создайте файл `main.py`, вставьте туда код игры и добавьте в папку проекта ваши ассеты (папки `images`, `music`, `levels`).
3. Нажмите `Shift + F10` для запуска, нажимайте **Start** и проверяйте игру на ошибки!
